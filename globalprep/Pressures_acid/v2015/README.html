Ocean Acidification Pressures Layer

Raw data provided by Ivan Lima from Woods Hole. This data is an updated to the work done by Feely, Doney and Cooley and published in 2009.

Monthly aragonite saturation state data provided for years 1880-1889 and 2005-2014 in NetCDF format with a resolution of about 1 degree.

Procedure

  1. oa_dataprep.r turns NetCDF raw files into .tifs and has 2 functions:
    1. calculates average annual aragonite saturation state for each year
    2. calculates the average decadal saturation state

Example data

  1. create_global_oa_pressures_layer.R does the following:

    1. Takes each of the 10 raster layers produced in step 1, and subtract the historical global mean (produced in step 1) to create 10 new raster layers (one for each year) with values equal to the change in aragonite saturation state
    2. All values that are 0 and below are set to 0
    3. Finds the maximum value across all 10 raster layers produced in step a
    4. Multiply this maximum value by 110% to get the reference point
    5. Divide all raster layers by reference point
    6. Interpolates values to the coast (for all 10 raster layers)
    7. Resamples to 1km for the final output raster layer (for all 10 raster layers)
    NOTE: reference point is 0.9259247 x 1.1 = 1.0185
